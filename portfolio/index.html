Data Mining Portfolio

by Levente Sipkei

This is a compilation of work that was done in Data Mining. The techniques and the strategies used to learn more about large amounts of data as well as discover useful patterns or correlations. The information and strategies in this portfolio are all described in detail along with sample of code. All the word was done in correlation of the class "Data Mining" with instructor: Yong Bakos. The primary sources of text for this portolio is "Collective Intelligence" by Toby Segram as well as "Introduction to Data Mining" by Pang-Ning Tan, Michael Steinbach and Vipin Kumar.
